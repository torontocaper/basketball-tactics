## Project Organization

There are four (for now) main folders:

### Builds

Exported, playable versions of the game. 

### Docs

Contains the Game Design Document, screenshots, itch.io cover images, Github social preview etc. 

### Experiments

A playground for new ideas. Messy, disorganized, prone to breakage. 

### Main

What other projects might call `src`; where the main, stable game project lives.

Scenes and scripts that 'work' will be migrated here from `experiments`.
